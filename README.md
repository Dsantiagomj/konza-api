# Konza

This is an API for manage simple tasks

[More information](https://github.com/dsantiagomj/konza-api)

## Instalation

1. `git clone https://github.com/dsantiagomj/konza-api`

2. `cd konza-api`

3. `npm i`

4. `npm start`

## Development

`npm run dev`
